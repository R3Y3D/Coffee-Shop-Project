# ☕ Sip & Scroll Website - Setup Guide

Welcome to the setup guide for the **Sip & Scroll** website. It now includes both a marketing landing page and a coffee ordering experience that works well on phones and laptops.

---

## 📁 Project Files
- [index.html](index.html) — the main landing page for the brand
- [order.html](order.html) — the coffee ordering page with login and cart flow
- [logo.jpeg](logo.jpeg) — the brand logo used across the pages

---

## 🛠️ How to Run the Website

### Step 1: Open the Main Page
1. Open [index.html](index.html) in your browser.
2. From there, click **Order Coffee** to open the ordering experience.

### Step 2: Use the Ordering Page
1. On [order.html](order.html), sign in with:
   - Username: `admin`
   - Password: `admin`
2. Browse the coffee menu, add drinks to your order, and place your order.
3. Your last ordered drink will be remembered and shown as a recommendation.

### Step 3: Share the Project
Keep [index.html](index.html), [order.html](order.html), and [logo.jpeg](logo.jpeg) in the same folder so the images and links work correctly.

---

## 🌐 Notes
- The site uses Tailwind CSS and Font Awesome from the internet, so an internet connection is recommended.
- The ordering page stores the last order locally in the browser for the previous-order suggestion feature.

---

## 🎨 Design Notes
- The color palette uses a warm coffee brown theme inspired by the logo.
- The layout is responsive and works on both mobile and desktop screens.
