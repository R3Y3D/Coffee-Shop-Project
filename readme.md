# ☕ Sip & Scroll Website - Setup Guide

Welcome to the official setup guide for the **Sip & Scroll** marketing landing page! This website was built based on your app concepts, logo color palette, and local café discovery data. 


---

## 🛠️ Step-by-Step Guide to Run the Website

### Step 1: Save the Code File
1. Right-click on your desktop or inside a folder where you want to keep the project.
2. Select **New** ➡️ **Text Document** (on Windows) or create a blank text file (on Mac).
3. Copy the entire HTML code provided in the previous message.
4. Paste the code into your new text document.
5. Save the file and rename it exactly to: `index.html`
   * *Note for Windows users:* Make sure the file extension changes from `.txt` to `.html`. If it still says `index.html.txt`, make sure you check "File name extensions" under the View tab in File Explorer to remove the hidden `.txt`.

### Step 2: Add your Brand Logo Image
For the branding and logo images to display properly inside the header and footer:
1. Locate your logo image asset file named **`image_7a7285.jpg`**.
2. Drag or move that image file into the **exact same folder** where you just saved your `index.html` file.
3. *Crucial:* Ensure the file name matches perfectly (`image_7a7285.jpg`). If it's a `.png` or named differently, change its name to match or the website won't find it.

### Step 3: Open the Website in a Browser
1. Double-click your newly created **`index.html`** file.
2. It will automatically launch and display beautifully in your default web browser (such as Google Chrome, Apple Safari, Microsoft Edge, or Mozilla Firefox).
3. **That's it!** You can scroll through the interactive sections, packages, and membership blocks.

## 🌐 How to Share it with Others
Since this is a standalone file that relies on external styling engines (Tailwind CSS and FontAwesome Icons loaded via the internet), make sure of the following if your friend wants to test it on their machine:

1. **Internet Connection:** Ensure the computer is connected to the internet so that fonts and layout designs load correctly on launch.
2. **Send as a Folder:** If you are sending this to your friend over WhatsApp, Discord, or Email, make sure to send **both** the `index.html` file and the `logo.jpeg` logo file together (ideally compressed into a `.zip` folder). Tell them to keep both files in the same directory!
